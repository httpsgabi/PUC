Descrição

Esse repositório foi criado para armazenar as atividades feitas durante o curso de Sistemas de Informação - PUC, no primeiro semestre, as matérias: Algoritmos e Técnicas de Programação(ATP), com foco na introdução à programação utilizando a linguagem C#, e Desenvolvimento de Interfaces Web(DIW), que possui foco no desesnvolvimento WEB utilizando HTML5, CSS3, e Javascript.
